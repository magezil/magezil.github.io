Journey into Programming

I first learned to code in high school using Java. I quickly got hooked on programming and joined the Computer Science Club. I learned that I love collaborating to solve interesting problems.

The thing I enjoy most about programming is tackling problems and figuring out how to use the available tools to solve the problem. For example, I love exploring and discovering solutions to the Project Euler problems. If I get stuck, I research the problem to discover more properties of the math and software concepts related to the problem. I also talk to other people around me about these problems to check if my understanding is correct or if there is another way to get to the solution.
